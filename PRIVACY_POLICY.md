# Privacy Policy for Prompt Tracer

**Last Updated: December 2024**

## Overview

Prompt Tracer ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how our browser extension collects, uses, and protects your information.

## Information We Collect

### Information We DO NOT Collect
- **No Personal Data**: We do not collect, store, or transmit any personal information
- **No Prompt Content**: Your prompts are never sent to external servers
- **No Usage Analytics**: We do not track your browsing habits or extension usage
- **No API Keys**: Your API keys (if provided) are stored locally on your device only

### Information We Store Locally
- **Prompt History**: Your analyzed prompts are stored locally in your browser
- **Performance Metrics**: Scoring and analytics data stored on your device
- **Settings**: Your extension preferences stored locally
- **API Keys** (Optional): If you choose to use premium features, your API keys are stored locally

## How We Use Your Information

### Local Processing Only
- **Prompt Analysis**: All analysis happens on your device using local algorithms
- **Performance Tracking**: Metrics are calculated and stored locally
- **Optimization**: Prompt improvements are generated using local rules and your own API keys

### Optional Premium Features
- **AI Optimization**: If you provide an API key, it's used to call OpenAI's API directly from your browser
- **Your Control**: You control whether to use premium features and can disable them anytime

## Data Storage and Security

### Local Storage
- All data is stored locally in your browser using Chrome's storage API
- Data never leaves your device unless you explicitly export it
- You can clear all data at any time through the extension settings

### API Key Security
- API keys are stored locally in encrypted browser storage
- Keys are only used for direct API calls from your browser
- We never have access to your API keys or API usage

## Third-Party Services

### OpenAI API (Optional)
- Only used if you provide your own API key
- Calls are made directly from your browser to OpenAI
- We do not intercept or store API responses
- You are responsible for API usage costs

### No Other Third Parties
- We do not use analytics services
- We do not use tracking pixels
- We do not share data with advertisers

## Your Rights and Choices

### Data Control
- **Export Data**: You can export your prompt history at any time
- **Clear Data**: You can delete all stored data through settings
- **Disable Features**: You can disable any feature you don't want to use

### Privacy Settings
- **Auto Analysis**: Can be disabled in settings
- **Data Storage**: Can be disabled to prevent local storage
- **Premium Features**: Can be disabled to prevent API usage

## Data Retention

### Local Data
- Prompt history is kept for the last 100 prompts by default
- You can change this limit or disable storage entirely
- Data persists until you clear it or uninstall the extension

### API Usage
- No API usage data is stored by us
- Check your OpenAI dashboard for API usage information

## Children's Privacy

Prompt Tracer is not intended for children under 13. We do not knowingly collect personal information from children under 13.

## Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by:
- Updating the "Last Updated" date at the top of this policy
- Posting the new policy in the extension

## Technical Implementation

### Browser Permissions
- **activeTab**: To analyze prompts on AI platforms
- **storage**: To save your preferences and prompt history locally
- **scripting**: To inject analysis functionality
- **downloads**: To export your data

### Data Flow
1. You type a prompt on an AI platform
2. Extension analyzes it locally on your device
3. Results are stored locally (if enabled)
4. Optional: Premium optimization using your API key
5. No data is transmitted to our servers
